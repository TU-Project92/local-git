# local-git

## 🚀 Setup Instructions

### Frontend

1. Navigate to the frontend directory:

   ```
   PS C:\Users\XXX\Desktop\local-git\frontend\frontend>
   ```

2. Install dependencies (if not already installed):

   ```
   npm i
   ```

3. Start the frontend:

   ```
   npm run start-noflp
   ```

### Backend

- Run the backend by starting the `BackendApplication`.

---

✅ Make sure both frontend and backend are running for the app to work properly.
